# Memory Specifications

Memory specs define selectable memory backends for agents.

## Required Fields

- `id`
- `version`
- `name`
- `description`

## Typical Fields

- `enabled`
- backend-specific configuration fields
- `tags`

## Notes

- Use memory IDs in agent `memory` fields.
- Keep backend selection explicit and versioned in catalogs.
